# Problem Statement: ISSR (Task 4 - FOA Ingestion)

## The Core Problem
Funding Opportunity Announcements (FOAs) are distributed across many agencies (e.g., Grants.gov, NSF, NIH), vary widely in file format and structure, and require significant manual effort to interpret. Research development teams often lose critical time and opportunities manually finding, parsing, and categorizing these FOAs to circulate to relevant investigators.

## The Goal
This project aims to build an open-source, automated pipeline that ingests FOAs from public sources, extracts structured fields (Title, Agency, Open/Close dates, Award ranges), and applies ontology-based semantic tags (Domain, Methodology, Population). This queryable funding intelligence database will support downstream institutional research discovery and automated grant matching.
